HTML là gì

HTML là viết tắt của cụm từ HyperText Markup Language, là ngôn ngữ đánh dấu siêu văn bản dùng để tạo ra các trang web. HTML được sử dụng để tạo ra cấu trúc của trang web, xác định các phần tử trên trang web, hiển thị nội dung trên trang web, tạo liên kết giữa các trang web,...

Javascript là gì

Javascript là ngôn ngữ lập trình dùng để tạo ra các trang web tương tác. Javascript được sử dụng để xử lý sự kiện, thay đổi nội dung trên trang web, kiểm tra dữ liệu nhập vào từ người dùng, tạo hiệu ứng chuyển động, tạo ra các ứng dụng web,...

css là gì

CSS là viết tắt của cụm từ Cascading Style Sheets, là ngôn ngữ định dạng dùng để tạo ra các trang web đẹp mắt. CSS được sử dụng để định dạng các tài liệu HTML, xác định kiểu chữ, màu sắc, khoảng cách giữa các phần tử, kích thước của các phần tử, vị trí của các phần tử, hiệu ứng chuyển động, hiển thị trên các thiết bị khác nhau,...

Có cần phải có css không

Không, bạn không cần phải có CSS để tạo ra một trang web. Tuy nhiên, nếu bạn muốn trang web của mình trở nên đẹp mắt, chuyên nghiệp và thân thiện với người dùng thì bạn cần phải sử dụng CSS.

Cách sử dụng CSS

Có 3 cách để sử dụng CSS:

  • Sử dụng CSS inline: CSS được viết trực tiếp trong thẻ HTML(inline).
  • Sử dụng CSS trong thẻ style: CSS được viết trong thẻ style trong phần head của trang web(internal).
  • Sử dụng CSS trong file riêng: CSS được viết trong file riêng và được gọi vào trong file HTML(external).
